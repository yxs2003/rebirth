关于 Rebirth 主题

项目介绍

Rebirth 是一个多风格可自选的自适应网站主题，基于 Flask 框架开发，不依赖任何 CMS，可直接运行。它的特点是 UI 自适应，支持多种风格切换，包括圆角拟态风格、老式报纸杂志风格和传统扁平风格。同时，它还支持黑夜/白天模式切换，为用户提供舒适的浏览体验。

Rebirth 主题适合各类网站使用，无论是个人博客、企业网站还是内容平台，都能通过简单的配置满足需求。它的设计理念是简洁、灵活、易用，让没有编程经验的用户也能轻松搭建出专业的网站。

核心功能

🎨

多风格主题

提供三种不同风格的主题，用户可以根据喜好随时切换。

📱

响应式设计

完美适配各种设备尺寸，从手机到桌面电脑。

🌓

黑夜/白天模式

支持手动切换或跟随系统设置自动切换显示模式。

⚙️

完整后台管理

提供直观的后台管理界面，轻松管理网站内容和设置。

📝

丰富内容模块

包含幻灯片、文章、合辑、公告等多种内容模块。

🔌

模块化结构

采用模块化设计，便于扩展和自定义开发。

技术栈

后端

  • Python 3.8+
  • Flask 框架
  • SQLAlchemy ORM
  • MySQL 数据库
  • RESTful API

前端

  • HTML5 / CSS3
  • JavaScript (ES6+)
  • 响应式设计
  • CSS 变量
  • Flexbox / Grid 布局
